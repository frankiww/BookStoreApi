# WebService: BooksApi
- Три точки доступа: получить все книги, получить книгу по айди, создать книгу
- Конвейер из трех обработчиков: логирование, унификация исключений, замер времени
- Ответы об ошибках содержат идентификатор запроса

## Установка и запуск
### 1. Установить зависимости
```
npm install
```

### 2. Запустить сервер
```
node index.js
```
## Точки доступа
### 1. Получить список всех книг
```
GET /api/items
```
```
curl http://localhost:8000/api/items
```
### 2. Получить книгу по айди
```
GET /api/items/:id
```
```
curl http://localhost:8000/api/items/1
```
### 3. Добавить книгу
```
POST /api/items
```
```
curl -X POST http://localhost:8000/api/items \
-H "Content-Type: application/json" \
-d '{"title":"Антрополог на Марсе", "author": "О. Сакс", "price":30}'
```
